# ブラケット編集：青色発光部の複数選択と一括作図

本ドキュメントでは、ブラケット編集（選択モード）で既存の青色発光部（柱）を複数選択し、Enterキーで方向と寸法の選択カード（既存UI）を用いて、複数箇所へ同時にブラケットを作図する手順を説明します。

## 概要
- 編集モードを「ブラケット」に切り替えます。
- 選択モード（通常／投げ縄／一括）時に、青色発光中の柱をクリックして複数選択できます。
- Enterキーを押すと「ブラケットの一括設定」カード（方向・寸法）を表示します。
- 「作図」を押すと、選択した柱位置に対して同じ方向・寸法のブラケットを一括作図します（同一点・同方向に既存がある場合は更新）。
- 一括メニュー（Layersアイコン）内の「数量」からも、直前の青色発光部の選択状態があれば同じカードを開けます。

## 操作手順
1. 左サイドバーのモードタブで「編集」を選択し、編集対象を「ブラケット」にします。
2. モードタブの選択モード（ポインタ／投げ縄／一括）を有効にします。
3. キャンバス上の青色発光中の柱を複数クリックして選択（トグル）します。
4. Enterキーを押すと、「ブラケットの一括設定」カードが表示されます。
5. 方向（→/↓/←/↑）と寸法（355/600/750/900）を選択し、「作図」を押します。
   - 既に同一点・同方向のブラケットがある場合は新規追加せずに上書きします。
   - 600/355 の場合は `bracketSize`（W/S）も自動設定します。
6. 完了後は選択状態が解除され、カードは閉じます。

## ショートカットと留意事項
- Enter: 一括設定カードを開く／カード表示中は「作図」を実行。
- Esc: カードを閉じる。
- 既存の単体カード（クリックで表示）とスタイル・操作感を統一しています。
- 一括メニューの「数量」は、選択対象に柱が含まれる場合は本カード（方向・寸法）を優先表示、ブラケット選択のみの場合は従来の数量カードを表示します。

## 実装ポイント（参考）
- `ScaffoldRenderer.tsx`：柱クリック時、ブラケット編集＋選択モードではトグル選択に変更。
- `CanvasStage.tsx`：一括表示時に柱選択が含まれる場合は `BracketConfigCardBulk` を表示、ブラケットのみの場合は `BracketQuantityCardUnified` を表示。
- `BracketConfigCardBulk.tsx`：方向・寸法の一括適用（同一座標・同方向は更新、未存在は新規作図）。
